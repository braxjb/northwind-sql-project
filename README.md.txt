# 🧾 Northwind SQL Project

This project uses the Northwind sample database to explore and demonstrate SQL skills, including data retrieval, joins, aggregations, and optimization. It serves as a showcase of practical SQL queries applied to a real-world business scenario.

## 📁 Project Structure

Northwind-SQL-Project/ │ ├── README.md ├── queries/ │ ├── 01_basic_selects.sql │ ├── 02_joins.sql │ ├── 03_aggregations.sql │ ├── 04_subqueries_cte.sql │ ├── 05_window_functions.sql │ └── 06_tcl_dcl.sql └── screenshots/ └── sample_output.png
`

## 💡 What You’ll Learn

- Writing SQL queries from beginner to advanced
- Using JOINs to combine related tables
- Working with GROUP BY, HAVING, and aggregate functions
- Using subqueries and common table expressions (CTEs)
- Exploring window functions like `ROW_NUMBER()`, `RANK()`, `LEAD()`, etc.
- Transaction Control (TCL) and Data Control (DCL) statements

## 📊 Example Queries

- List top 5 products by sales
- Identify the most valuable customers
- Calculate average order value per region
- Find repeat customers using window functions

## 🛠 Requirements

- Microsoft SQL Server or any SQL-compatible system
- Northwind Database (can be restored from `.bak` or `.sql` file)
